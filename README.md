# UPI-QR-Code-Generator-using-Python
 Developed a Python-based application to generate UPI payment QR codes 
 Used qrcode library to convert UPI URLs into scannable QR images 
 Enabled compatibility with PhonePe, Paytm, and Google Pay 
 Improved payment accuracy and reduced manual entry errors

MINI PROJECT DOCUMENTATION
Title

UPI QR Code Generator Using Python

1. Introduction

Digital payments have become an essential part of everyday transactions. UPI (Unified Payments Interface) enables instant money transfer using mobile devices. This project focuses on generating a UPI payment QR code using Python, which can be scanned by any UPI-enabled application.

2. Problem Statement

Manual sharing of UPI IDs may lead to errors and inconvenience. A simple, fast, and reliable method is required to generate a scannable QR code for UPI payments.

3. Objective

To generate a UPI payment QR code

To automate UPI payment initiation

To reduce manual errors in payment details

4. Tools & Technologies Used

Programming Language: Python

Library: qrcode

Platform: Any OS with Python installed

5. System Requirements
Hardware:

Computer / Laptop

Camera-enabled smartphone (for scanning)

Software:

Python 3.x

qrcode library

6. Methodology

User enters UPI details

Program generates a UPI payment URL

QR code is created from the URL

QR image is saved

User scans QR to complete payment

7. Conclusion

This project successfully demonstrates how Python can be used to create a UPI QR code. It provides a fast, secure, and efficient way to receive payments digitally.
